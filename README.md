>## **Git Basic Commands**
>
>>**1.Clone**
>>
>>> **Syntax**
>>>>--> **git clone \<repo-url\>**
>>>- **Note :** clone can be achieved using HTTPs || SSH method.
>>>>
>>> **Example**
>>>>- \>**git clone "https://github.com/Infinite-Learner/CD.git"** # **https** method.
>>>>- \>**git clone "git clone git@github.com:Infinite-Learner/CD.git"** # **SSH** method.
>>>>
